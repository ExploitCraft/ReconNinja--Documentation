# ReconNinja Docs

A single-file static documentation page for [ReconNinja](https://github.com/ExploitCraft/ReconNinja) — the 14-phase automated reconnaissance framework.

## What this is

`index.html` is a self-contained HTML documentation site. No build tools, no dependencies, no npm — just open it in a browser or serve it as a static page.

## Deploy

**GitHub Pages** — rename to `index.html`, push to a `docs/` branch or repo, enable Pages in settings.

**Cloudflare Pages** — connect repo, set output to `/`, no build command needed.

**Local** — just open the file directly in any browser.

## Stack

Pure HTML + CSS + JS. Zero dependencies. Zero frameworks.
